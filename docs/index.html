Installation

Android APK

TerraRun is distributed as an Android APK. Follow these steps to install:

  1. Download the latest APK from the Downloads page
  2. Go to Settings → Security → Unknown Sources
  3. Enable installation from unknown sources
  4. Tap the downloaded APK file
  5. Grant requested permissions
  6. Done! Launch TerraRun from your app drawer
Note: Different Android versions may have slightly different settings. On Android 8+, you can restrict unknown app installs to specific apps (recommended).

Quick Start

Get started with your first run in 3 steps:

1

Open the App

Launch TerraRun and allow GPS + location permissions when prompted

2

Start Recording

Tap the "Start Run" button. Wait 30 seconds for GPS lock, then begin running

3

Finish & Save

Tap "Stop" when done. Review your stats and save your run

Requirements

Minimum Requirements

  • Android: 8.0 (API 26) or higher
  • RAM: 2GB minimum
  • Storage: 50MB available space
  • GPS: Hardware GPS receiver

Recommended Specifications

  • Android: 10 or higher
  • RAM: 4GB or more
  • Storage: SSD or modern storage (for smooth performance)
  • GPS: A-GPS capable device

Recording Runs

TerraRun captures comprehensive data about every run you record:

What Gets Recorded

  • GPS route with precise coordinates
  • Distance, duration, and pace
  • Elevation gain/loss
  • Heart rate (if compatible device paired)
  • Cadence and stride metrics
  • Weather conditions at time of run

During Your Run

While recording, TerraRun displays real-time stats:

  • Current Pace: Your speed at this moment
  • Average Pace: Your average speed since start
  • Distance: Total kilometers/miles covered
  • Time Elapsed: Running duration
  • Elevation: Height gained/lost

GPS Tracking

TerraRun uses advanced GPS technology for accurate route tracking.

GPS Accuracy

Our typical accuracy is ±3-5 meters in open areas. In urban canyons or dense forests, this can degrade to ±10-20 meters.

Tips for Better Accuracy

  • Ensure you have clear sky view when starting
  • Wait for GPS lock (indicated by a steady signal icon)
  • Avoid tunnels and dense urban areas when possible
  • Keep your device in view during the run (in hand or armband)
  • Use A-GPS capable devices for faster satellite acquisition

Battery Impact

GPS is power-intensive. TerraRun's optimized mode achieves 6+ hours of continuous tracking on most modern phones.

Offline Maps

Download maps for offline use in areas with poor connectivity.

Downloading Maps

  1. Open the Maps section in TerraRun
  2. Find your region or search by location
  3. Tap "Download" next to the map
  4. Wait for download to complete
  5. Maps are now available offline
Storage Note: Offline maps range from 50MB-500MB depending on region. Ensure sufficient storage available.

Managing Maps

Maps are stored in Settings → Storage → Offline Maps. You can:

  • View map size and coverage area
  • Delete maps to free up space
  • Update maps to the latest versions

Pace Coaching

Our AI-powered pace coach adapts in real-time to terrain and conditions.

How It Works

The pace coach analyzes:

  • Current elevation and slope
  • Your target pace settings
  • Heart rate zones (if available)
  • Running history and fitness level

It then provides real-time guidance to adjust your pace for optimal performance.

Configuring Pace Zones

Set your pace zones in Settings → Training → Pace Zones:

  • Easy: Recovery and base building
  • Steady: Conversational pace
  • Tempo: Challenging but sustainable
  • Threshold: Hard effort
  • VO2 Max: Maximum effort intervals

Data Export

Export your run data in multiple formats for use with other apps.

Supported Formats

  • GPX: Universal GPS exchange format (recommended)
  • FIT: Garmin's proprietary format
  • TCX: Training Center XML
  • JSON: For developers and custom integrations

How to Export

  1. Open a completed run
  2. Tap the "Share" button
  3. Select "Export as GPX" (or other format)
  4. Choose destination or app

GPX Import

Import routes or previous runs in GPX format.

Importing Routes

  1. Go to Routes → Import
  2. Select a GPX file from your device
  3. Review the route details
  4. Tap "Import" to save

Importing Past Runs

You can import historical run data from other apps:

  1. Export your runs as GPX from your previous app
  2. Use the History → Import feature
  3. Select the GPX file
  4. Confirm import (data will be added to your history)

Privacy & Security

Data Privacy

TerraRun respects your privacy:

  • All GPS data is stored locally on your device
  • Cloud sync is optional and encrypted end-to-end
  • We never sell user data
  • Your location data is private by default

Permissions Explained

GPS/Location
Required for route tracking and mapping
Storage
For saving runs and offline maps
Health/Fitness
Optional - to sync with health apps
Bluetooth
Optional - to connect to heart rate monitors

API Reference

TerraRun provides REST API endpoints for developers.

Base URL

https://api.terrarun.dev/v1

Authentication

Include your API key in headers:

Authorization: Bearer YOUR_API_KEY

Endpoints

GET
/users/{id}/runs

Get all runs for a user

POST
/runs

Create a new run

GET
/runs/{id}

Get run details

GET
/routes

List public routes
